# PhotoSearch
Example Android TV App - Contains a Single Use Case to fetch and search through a Flickr Feed

# What I used on this project
- [Jetpack Compose](https://developer.android.com/jetpack/compose/tutorial)
- [Jetpack Paging](https://developer.android.com/topic/libraries/architecture/paging/v3-overview)
- [Jetpack Lifecycle](https://developer.android.com/topic/libraries/architecture/lifecycle)
- [Retrofit](https://square.github.io/retrofit/)
- [Okhhtp](https://github.com/square/okhttp)
- [Gson](https://github.com/google/gson)
- [Coil](https://coil-kt.github.io/coil/)
